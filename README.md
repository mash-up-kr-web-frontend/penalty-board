# 벌금 현황판

## 벌금 종류

1. 결석 벌금: React 스터디 불참할 시 벌금 2,000원 (JS 스터디는 결석 벌금 따로 없음)
2. 과제 벌금: 과제 미제출 시 벌금 2,000원

## 벌금 면제 기준

- 집안 경조사로 인한 불참인 경우에만 결석 벌금 면제. **과제 벌금은 면제되지 않음.**
- **친척 또는 지인의 경조사는 집안 경조사의 범위에 포함되지 않음.** 오로지 가족까지만 집안 경조사로 인정
